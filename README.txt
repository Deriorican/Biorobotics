Organisation du Git :

Bibliographie :

Ensemble des références bibliographiques utilisées dans le cadre du projet. Chaque document est accompagné d'une review, un résumé du document mettant en lumière les points
importants du document. Le but est de savoir en un coup d'oeil quelle est l'utilité du document pour le projet, ce qui servira à la sélection des documents pertinents par après.
Se trouve également dans le dossier un document listant l'ensemble des références.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Codes :

Dossier rassemblant les différents codes informatiques servant à la production de figures, simulations ou autres.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Figures :

Dossier rassemblant les productions des codes et qui seront (ou non) utilisées dans le rapport.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Présentation :

Slides et autres supports utiles à la présentation du projet devant la classe.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Rapport : 

Ensemble des codes Latek formant le rapport final
